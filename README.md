<h1>Coin App MVC-Formik</h1>

<p>This React project, which provides real-time tracking of crypto currencies, was developed based on MVC architecture. Formik and Yup libraries were used for form management, and Chart.js was used to create tables.</p>

<h3>Used Libraries</h3>

<p>bootstrap, formik, yup, react-router-dom, axios, chart.js, react-chartjs-2, react-icons, millify.</p>
